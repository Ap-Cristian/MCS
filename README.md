# Metal Cooling Simulator (MCS)
Web-based Metal Cooling Simulator made form scratch using WebGPU.
# Chimera & Equinox
Both apps use the same command set:

```
npm run build
```
to bundle your application.

To run just use 
```
npm run start
```
after installing the required dependencies
