# Metal Cooling Simulator (MCS)
Web-based Metal Cooling Simulator made form scratch using WebGPU.
# Equinox

Firstly, install the required dependencies using
```
npm i
```
then you can build the app using.
```
npm run build
```
To run just use 
```
npm run start
```
